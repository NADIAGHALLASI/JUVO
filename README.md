# JUVO

A modern and responsive front-end web application for a juice and healthy drinks shop, built with HTML, CSS, and JavaScript.

JUVO provides an interactive shopping experience with a clean user interface, user authentication, product browsing, and shopping cart functionality.

---

## Features

- User Authentication (Login & Registration)
- Product Catalog
- Shopping Cart
- Responsive Design
- Local Storage
- Modern UI/UX
- Interactive User Experience

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## Project Goal

This project was developed to improve my front-end development skills by building a responsive shopping interface using Vanilla JavaScript without external frameworks.
